# LabSim — Healthcare Data Science on Synthetic Data

Nine end-to-end machine learning projects built on a **fully synthetic
healthcare dataset** engineered to mirror the operational patterns of real
diagnostic laboratories.

**Live site:** https://labsim-ds.vercel.app

## The story

I lead data operations at a YC-backed healthcare diagnostics SaaS company,
where I built an internal training platform for our data science intern
cohort. Interns learn fastest on data that behaves like production — but real
healthcare data cannot simply be handed around.

LabSim is the public twin of that system. Every record is generated by
`scripts/generate_synthetic.py` — a pipeline that reproduces realistic
distributions, five distinct client personalities, and the exact
data-quality problems that break naive analyses in production:

- Null join keys (~15% of service→order links)
- Partial timestamp coverage (~66%)
- Batch-locked completion times
- Zero-value insurance-style records
- Broken reference ranges
- Multilingual free-text radiology impressions (Bahasa Indonesia)
- Financial anomalies **with known ground truth** for measurable detection

No real patient, physician, facility, or client appears anywhere in this
repository.

## The nine projects

| # | Project | Difficulty | Core technique |
|---|---------|-----------|----------------|
| 1 | Abnormal Lab Result Predictor | Beginner | LogReg / RF / XGBoost |
| 2 | Turnaround Time Predictor | Beginner | Regression |
| 3 | Patient Segmentation | Intermediate | K-Means + PCA |
| 4 | Test Bundle Analysis | Beginner | Apriori |
| 5 | Revenue Anomaly Detection | Intermediate | Isolation Forest |
| 6 | Support Ticket Intelligence | Intermediate | NLP classification |
| 7 | Clinical Notes NLP | Advanced | LDA + NER |
| 8 | Multilingual Radiology Parser | Advanced | Negation-aware NLP |
| 9 | Cross-Client Benchmarking | Intermediate | ANOVA + normalization |

## Stack

Next.js 15 (App Router) · TypeScript · Tailwind CSS v4 · Recharts ·
Framer Motion · Python (pandas, numpy, faker) · Vercel

## Quick start

```bash
pnpm install
python scripts/generate_synthetic.py   # generates data/synthetic/
pnpm dev                               # localhost:3000
```
